هدف پروژه

در این پروژه هدف ما راه اندازی یک وب سرور بر روی یک embeded سیستم مبتنی بر پردازنده ARM بود.


این پروژه توسط سید فرشاد کاظمی و کیوان قارونی صفار در دانشگاه فردوسی مشهد - دانشکده مهندسی زیر نظر استاد گرانقدر دکتر حمید نوری با مشاوره مهندس هاشم قربان پناه انجام گرفته است.
